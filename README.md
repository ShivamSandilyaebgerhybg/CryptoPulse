# CryptoPulse
This is a "front-end only" Project which tracks crypto prices in real time.
It refreshes the data every 15 seconds including the price , graph , ranking , growth , etc.
As it's a "front-end only" Project we're using API from "CorinGecko" for the realtime updates.
It has 2 Prices "USD" ,& "INR".
It has a dark-mode and light-mode feature also so that people can use it according to the surrounding light.
